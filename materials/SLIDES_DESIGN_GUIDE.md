# Slide Design Guide

Format
- 16:9, dark theme, glassmorphism accents
- Fonts: Space Grotesk (display), Inter (body)
- Color: violet/indigo gradients matching app UI

Layout Tips
- Max 3 bullets per slide, use visuals/screenshots where possible
- Keep consistent margins and grid
- Show product first, text second

Assets to include
- Screenshots: Feed, Trends, NFT, Upload, Player bar, Statistics
- Small architecture diagram (boxes: React, Zustand, IPFS, ethers, Supabase)

Export
- Export both PDF and PNG (slides) for sharing on marketplaces
