# Video Capture Guide

Tools
- macOS: QuickTime or CleanShot X
- Windows: Xbox Game Bar or OBS Studio

Settings
- 1080p (1920x1080), 30fps, H.264 mp4
- Cursor highlight disabled; smooth mouse movement

Tips
- Prepare demo data (SoundHelix URLs already present)
- Practice flow using DEMO_SCRIPT.md and STORYBOARD.md
- Keep VO short; add captions if possible
